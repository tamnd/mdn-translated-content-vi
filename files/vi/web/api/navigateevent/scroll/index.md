{{APIRef("Navigation API")}}

Phương thức `scroll()` của {{domxref("NavigateEvent")}} được dùng để kích hoạt thủ công hành vi cuộn do trình duyệt điều khiển, nếu bạn muốn thực hiện trước khi quá trình xử lý điều hướng hoàn tất.

## Cú pháp

```js
scroll();
```
