{{APIRef("Navigation API")}}

Thuộc tính `info` của {{domxref("NavigateEvent")}} trả về giá trị dữ liệu `info` được truyền bởi thao tác điều hướng khởi tạo, hoặc `undefined` nếu không có dữ liệu `info` nào được truyền.

## Giá trị

Bất kỳ giá trị JavaScript nào hoặc `undefined`.
