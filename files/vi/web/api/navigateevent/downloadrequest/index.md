{{APIRef("Navigation API")}}

Thuộc tính `downloadRequest` của {{domxref("NavigateEvent")}} trả về tên tệp được yêu cầu tải xuống trong trường hợp điều hướng tải xuống, hoặc `null` trong các trường hợp khác.

## Giá trị

Một chuỗi hoặc `null`.
