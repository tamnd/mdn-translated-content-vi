{{APIRef("Navigation API")}}

Thuộc tính `userInitiated` của {{domxref("NavigateEvent")}} trả về `true` nếu điều hướng được khởi tạo bởi người dùng, ngược lại trả về `false`.

## Giá trị

Một giá trị boolean.
