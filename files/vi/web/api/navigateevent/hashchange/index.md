{{APIRef("Navigation API")}}

Thuộc tính `hashChange` của {{domxref("NavigateEvent")}} trả về `true` nếu điều hướng là điều hướng fragment, tức là tới một định danh fragment trong cùng tài liệu, ngược lại trả về `false`.

## Giá trị

Một giá trị boolean.
