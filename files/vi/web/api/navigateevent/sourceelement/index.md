{{APIRef("Navigation API")}}

Thuộc tính `sourceElement` của {{domxref("NavigateEvent")}} trả về đối tượng {{domxref("Element")}} đại diện cho phần tử đã khởi tạo điều hướng, nếu có.

## Giá trị

Một đối tượng {{domxref("Element")}} hoặc `null`.
