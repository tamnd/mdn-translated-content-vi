{{APIRef("Web Components")}}

Phương thức **`delete()`** của giao diện {{domxref("CustomStateSet")}} xóa một giá trị đơn khỏi `CustomStateSet`.

## Cú pháp

```js-nolint
delete(value)
```

### Tham số

- `value`
  - : Giá trị cần xóa khỏi `CustomStateSet`.

### Giá trị trả về

Trả về `true` nếu `value` có trong `CustomStateSet`; ngược lại trả về `false`.
