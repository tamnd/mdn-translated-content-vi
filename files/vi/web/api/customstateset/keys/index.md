## Cú pháp

```js-nolint
keys()
```

### Tham số

Không có.

### Giá trị trả về

Một đối tượng iterator mới chứa các giá trị cho mỗi phần tử trong `CustomStateSet` đã cho, theo thứ tự chèn vào.
