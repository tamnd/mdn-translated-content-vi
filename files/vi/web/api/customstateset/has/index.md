{{APIRef("Web Components")}}

Phương thức **`has()`** của giao diện {{domxref("CustomStateSet")}} trả về {{jsxref("Boolean")}} xác nhận xem phần tử có mặt với giá trị đã cho hay không.

## Cú pháp

```js-nolint
has(value)
```

### Tham số

- `value`
  - : Giá trị cần kiểm tra trong đối tượng `CustomStateSet`.

### Giá trị trả về

Một {{jsxref("Boolean")}}, `true` nếu `value` tồn tại trong `CustomStateSet`.
