{{APIRef("Service Workers API")}}{{AvailableInWorkers("service")}}

Thuộc tính chỉ đọc **`frameType`** của giao diện {{domxref("Client")}} cho biết kiểu ngữ cảnh duyệt web của {{domxref("Client")}} hiện tại. Giá trị này có thể là một trong các chuỗi `"auxiliary"`, `"top-level"`, `"nested"` hoặc `"none"`.

## Giá trị

Một trong bốn chuỗi sau: `"auxiliary"`, `"top-level"`, `"nested"` hoặc `"none"`.
