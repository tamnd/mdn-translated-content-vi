{{ apiref("CSSOM") }}

Thuộc tính chỉ đọc **`TransitionEvent.pseudoElement`** là chuỗi bắt đầu bằng `'::'`, chứa tên của [phần tử giả](/en-US/docs/Web/CSS/Reference/Selectors/Pseudo-elements) mà hoạt ảnh chạy trên đó. Nếu transition không chạy trên phần tử giả mà trên phần tử, chuỗi rỗng: `""`.

## Giá trị

Một chuỗi.
