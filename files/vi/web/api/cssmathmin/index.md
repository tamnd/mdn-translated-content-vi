Giao diện **`CSSMathMin`** của [CSS Typed Object Model API](/en-US/docs/Web/API/CSS_Object_Model) biểu thị hàm CSS {{CSSXref('min','min()')}}. Nó kế thừa thuộc tính và phương thức từ lớp cha {{domxref('CSSNumericValue')}}.

## Hàm khởi tạo

- {{domxref("CSSMathMin.CSSMathMin", "CSSMathMin()")}} {{Experimental_Inline}}
  - : Tạo một đối tượng `CSSMathMin` mới.

## Thuộc tính phiên bản

- {{domxref('CSSMathMin.values')}} {{ReadOnlyInline}}
  - : Trả về một đối tượng {{domxref('CSSNumericArray')}} chứa một hoặc nhiều đối tượng {{domxref('CSSNumericValue')}}.
