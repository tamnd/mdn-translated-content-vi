{{ APIRef("Touch Events") }}

Thuộc tính chỉ đọc **`radiusY`** của giao diện {{domxref("Touch")}} trả về bán kính Y của hình elip gần nhất bao quanh vùng tiếp xúc với bề mặt cảm ứng. Giá trị tính bằng pixel CSS theo cùng tỷ lệ với {{ domxref("Touch.screenX") }}.

Kết hợp với {{ domxref("Touch.radiusX") }} và {{ domxref("Touch.rotationAngle") }}, giá trị này tạo thành hình elip xấp xỉ kích thước và hình dạng vùng tiếp xúc giữa người dùng và màn hình. Đây có thể là hình elip lớn thể hiện tiếp xúc từ đầu ngón tay, hoặc vùng nhỏ thể hiện đầu bút stylus.

## Giá trị

Một số.

## Ví dụ

Xem [ví dụ Touch.radiusX](/en-US/docs/Web/API/Touch/radiusX#examples) để biết cách dùng thuộc tính này.
