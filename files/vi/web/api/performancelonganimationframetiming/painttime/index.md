{{APIRef("Performance API")}}

Thuộc tính chỉ đọc **`paintTime`** của giao diện {{domxref("PerformanceLongAnimationFrameTiming")}} trả về {{domxref("DOMHighResTimeStamp","timestamp")}} khi giai đoạn kết xuất kết thúc và khung hoạt hình bắt đầu.

`paintTime` có tính tương thích rộng: Giá trị phải giống nhau trên các triển khai khác nhau.

## Giá trị

Một {{domxref("DOMHighResTimeStamp")}}.
