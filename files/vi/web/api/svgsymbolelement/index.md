{{APIRef("SVG")}}

Giao diện **`SVGSymbolElement`** tương ứng với phần tử {{SVGElement("symbol")}}.

{{InheritanceDiagram}}

## Thuộc tính phiên bản

_Giao diện này cũng kế thừa các thuộc tính từ giao diện cha, {{domxref("SVGGraphicsElement")}}._

- {{domxref("SVGSymbolElement.viewBox")}} {{ReadOnlyInline}}
  - : Một {{domxref("SVGAnimatedRect")}} tương ứng với thuộc tính {{SVGAttr("viewBox")}} của phần tử {{SVGElement("symbol")}} tương ứng.
- {{domxref("SVGSymbolElement.preserveAspectRatio")}} {{ReadOnlyInline}}
  - : Một {{domxref("SVGAnimatedPreserveAspectRatio")}} tương ứng với thuộc tính {{SVGAttr("preserveAspectRatio")}} của phần tử {{SVGElement("symbol")}} tương ứng.

## Phương thức phiên bản

_Giao diện này không triển khai phương thức riêng mà kế thừa các phương thức từ giao diện cha, {{domxref("SVGGraphicsElement")}}._
