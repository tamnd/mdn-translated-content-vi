{{ApiRef("HTML DOM")}}{{Deprecated_Header}}

Thuộc tính **`compact`** của giao diện {{domxref("HTMLDListElement")}} chỉ ra rằng khoảng cách giữa các mục danh sách nên được giảm. Cách xử lý chính xác thuộc tính `compact` phụ thuộc vào từng trình duyệt. Thay vì sử dụng thuộc tính này, hãy xem xét sử dụng CSS {{cssxref("line-height")}} thay thế.

Nó phản ánh thuộc tính [`compact`](/en-US/docs/Web/HTML/Reference/Elements/dl#compact) của phần tử {{HTMLElement("dl")}}.

## Giá trị

Giá trị boolean.
