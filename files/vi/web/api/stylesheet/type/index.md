{{APIRef("CSSOM")}}

Thuộc tính **`type`** của giao diện {{domxref("StyleSheet")}} chỉ định ngôn ngữ bảng định kiểu cho bảng định kiểu đã cho.

## Giá trị

Một chuỗi.

## Ví dụ

```js
myStyleSheet.type = "text/css";
```
