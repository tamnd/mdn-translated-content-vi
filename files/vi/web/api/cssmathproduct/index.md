{{APIRef("CSS Typed Object Model API")}}

Giao diện **`CSSMathProduct`** của [CSS Typed Object Model API](/en-US/docs/Web/API/CSS_Object_Model) biểu thị kết quả thu được khi gọi {{domxref('CSSNumericValue.add','add()')}}, {{domxref('CSSNumericValue.sub','sub()')}}, hoặc {{domxref('CSSNumericValue.toSum','toSum()')}} trên {{domxref('CSSNumericValue')}}. Nó kế thừa thuộc tính và phương thức từ lớp cha {{domxref('CSSNumericValue')}}.

## Hàm khởi tạo

- {{domxref("CSSMathProduct.CSSMathProduct", "CSSMathProduct()")}} {{Experimental_Inline}}
  - : Tạo một đối tượng `CSSMathProduct` mới.

## Thuộc tính phiên bản

- {{domxref('CSSMathProduct.values')}}
  - : Trả về một đối tượng {{domxref('CSSNumericArray')}} chứa một hoặc nhiều đối tượng {{domxref('CSSNumericValue')}}.
