Hàm khởi tạo **`CSSMathProduct()`** tạo một đối tượng {{domxref("CSSMathProduct")}} mới, nhân các đối số được truyền vào nó.

## Cú pháp

```js-nolint
new CSSMathProduct(args)
```

### Tham số

- `args`
  - : Danh sách các giá trị cho đối tượng {{domxref('CSSMathProduct')}} là số nguyên double hoặc {{domxref('CSSNumericValue')}}.
