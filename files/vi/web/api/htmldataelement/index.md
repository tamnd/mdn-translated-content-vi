{{APIRef("HTML DOM")}}

Giao diện **`HTMLDataElement`** cung cấp các thuộc tính đặc biệt (bên cạnh giao diện {{domxref("HTMLElement")}} mà nó cũng kế thừa) để thao tác các phần tử {{HTMLElement("data")}}.

{{InheritanceDiagram}}

## Thuộc tính instance

_Cũng kế thừa các thuộc tính từ phần tử cha, {{domxref("HTMLElement")}}._

- {{domxref("HTMLDataElement.value")}}
  - : Một chuỗi phản ánh thuộc tính HTML [`value`](/en-US/docs/Web/HTML/Reference/Elements/data#value), chứa dạng máy đọc của giá trị phần tử.

## Phương thức instance

_Không có phương thức cụ thể nào; kế thừa các phương thức từ phần tử cha, {{domxref("HTMLElement")}}._
