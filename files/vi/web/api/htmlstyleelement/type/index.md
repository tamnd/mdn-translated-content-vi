Thuộc tính **`HTMLStyleElement.type`** trả về kiểu của kiểu dáng hiện tại.
Giá trị phản ánh [thuộc tính `type` của phần tử HTML `<style>`](/en-US/docs/Web/HTML/Reference/Elements/style#type).

Các tác giả không nên sử dụng thuộc tính này hoặc phụ thuộc vào giá trị của nó.

## Giá trị

Các giá trị được phép là chuỗi rỗng hoặc so khớp không phân biệt hoa thường với "text/css".
