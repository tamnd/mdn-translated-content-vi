{{APIRef("WebRTC")}}

Thuộc tính **`password`** của từ điển **{{domxref("RTCIceParameters")}}** xác định password ICE mà cùng với {{domxref("RTCIceParameters.usernameFragment", "usernameFragment")}}, xác định duy nhất một phiên ICE trong toàn bộ thời gian của nó.

## Giá trị

Một chuỗi chứa password tương ứng với chuỗi `usernameFragment` của transport.
