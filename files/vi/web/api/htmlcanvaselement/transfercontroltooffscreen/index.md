{{APIRef("Canvas API")}}

**`HTMLCanvasElement.transferControlToOffscreen()`** chuyển quyền điều khiển sang một đối tượng {{domxref("OffscreenCanvas")}}, trên main thread hoặc trên worker.

## Cú pháp

```js-nolint
transferControlToOffscreen()
```
