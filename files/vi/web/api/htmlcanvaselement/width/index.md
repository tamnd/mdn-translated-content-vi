{{APIRef("Canvas API")}}

Thuộc tính **`HTMLCanvasElement.width`** là một số nguyên dương phản ánh thuộc tính HTML [`width`](/en-US/docs/Web/HTML/Reference/Elements/canvas#width) của phần tử {{HTMLElement("canvas")}} khi được hiểu theo CSS pixel. Khi thuộc tính không được chỉ định, hoặc được đặt thành giá trị không hợp lệ như số âm, giá trị mặc định `300` sẽ được dùng.

Việc đặt thuộc tính `width` sẽ đặt lại toàn bộ rendering context về trạng thái mặc định.
