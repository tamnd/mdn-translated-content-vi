{{APIRef("DOM")}}

Thuộc tính chỉ đọc **`assignedSlot`** của giao diện {{domxref("Text")}} trả về đối tượng {{domxref("HTMLSlotElement")}} liên kết với phần tử đó.

## Giá trị

Một {{domxref("HTMLSlotElement")}}, hoặc `null` nếu không có phần tử {{HTMLElement("slot")}} nào liên kết với nút văn bản.
