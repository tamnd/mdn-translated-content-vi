Thuộc tính chỉ đọc **`featurePolicy`** của giao diện {{domxref("Document")}} trả về giao diện {{domxref("FeaturePolicy")}} cung cấp một API đơn giản để kiểm tra [Permissions Policies](/en-US/docs/Web/HTTP/Guides/Permissions_Policy) được áp dụng cho một tài liệu cụ thể.

## Giá trị

Một đối tượng {{domxref("FeaturePolicy")}} có thể được sử dụng để kiểm tra các cài đặt Permissions Policy được áp dụng cho tài liệu.
