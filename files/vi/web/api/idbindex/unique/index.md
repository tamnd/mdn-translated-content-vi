Thuộc tính chỉ đọc **`unique`** của giao diện `IDBIndex` cho biết index này có cho phép các giá trị trùng lặp cho một khóa hay không.

## Giá trị

`true` nếu index là duy nhất; ngược lại là `false`.

## Ví dụ

```js
console.log(myIndex.unique);
```
