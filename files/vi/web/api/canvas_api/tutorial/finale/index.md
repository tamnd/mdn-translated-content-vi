## Thêm ví dụ và hướng dẫn

Có nhiều bản demo và giải thích bổ sung về canvas trên các trang web này:

- [Canvas Codepens](https://codepen.io/search/pens?q=canvas)
  - : Sân chơi dành cho lập trình viên frontend và trình biên tập mã chạy trong trình duyệt.
- [Phát triển trò chơi](/en-US/docs/Games)
  - : Chơi game là một trong những hoạt động phổ biến nhất trên máy tính. Các công nghệ mới liên tục xuất hiện để giúp tạo ra những trò chơi tốt hơn và mạnh mẽ hơn có thể chạy trong bất kỳ trình duyệt web tuân thủ tiêu chuẩn nào.

## API web khác

API này có thể hữu ích khi làm việc sâu hơn canvas và đồ họa:

- [WebGL](/en-US/docs/Web/API/WebGL_API)
  - : API nâng cao để dựng đồ họa phức tạp, bao gồm cả 3D.
- [SVG](/en-US/docs/Web/SVG)
  - : Đồ họa vector có thể co giãn cho phép bạn mô tả hình ảnh dưới dạng tập hợp các vector và hình dạng để chúng có thể co giãn mượt mà bất kể kích thước hiển thị.
- [Web Audio](/en-US/docs/Web/API/Web_Audio_API)
  - : Web Audio API cung cấp một hệ thống mạnh mẽ và linh hoạt để điều khiển âm thanh trên web, cho phép nhà phát triển chọn nguồn âm thanh, thêm hiệu ứng, tạo hình trực quan âm thanh, áp dụng hiệu ứng không gian như panning và nhiều hơn nữa.
