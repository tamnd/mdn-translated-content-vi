{{ApiRef("HTML DOM")}}{{Deprecated_Header}}

Thuộc tính **`compact`** của giao diện {{domxref("HTMLMenuElement")}} cho biết rằng khoảng cách giữa các mục danh sách nên được giảm. Cách xử lý chính xác thuộc tính `compact` phụ thuộc vào trình duyệt. Thay vì sử dụng thuộc tính này, hãy cân nhắc sử dụng CSS {{cssxref("line-height")}}.

Nó phản ánh thuộc tính [`compact`](/en-US/docs/Web/HTML/Reference/Elements/menu#compact) của phần tử {{HTMLElement("menu")}}.

## Giá trị

Một giá trị boolean.
