Getter **`item`** của giao diện {{domxref("SpeechGrammarList")}} là một getter tiêu chuẩn, cho phép lấy các đối tượng {{domxref("SpeechGrammar")}} riêng lẻ từ `SpeechGrammarList` bằng cú pháp mảng.

## Cú pháp

```js-nolint
item(index)
```

### Tham số

- `index`
  - : Chỉ mục của mục cần lấy.

### Giá trị trả về

Một đối tượng {{domxref("SpeechGrammar")}}.
