{{securecontext_header}}{{APIRef("Payment Request API")}}

Phương thức **`complete()`** của giao diện {{domxref("PaymentResponse")}} thông báo cho user agent rằng tương tác của người dùng đã kết thúc và giao diện còn lại có thể được đóng.

## Cú pháp

```js
complete(result);
```

## Tham số

- `result`
  - : Kết quả hoàn tất tùy chọn.
