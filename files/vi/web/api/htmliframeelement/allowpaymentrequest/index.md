Thuộc tính **`allowPaymentRequest`** của giao diện {{domxref("HTMLIFrameElement")}} trả về một giá trị boolean cho biết liệu [Payment Request API](/en-US/docs/Web/API/Payment_Request_API) có thể được gọi trên một iframe khác nguồn gốc hay không.

## Giá trị

Một giá trị boolean.
