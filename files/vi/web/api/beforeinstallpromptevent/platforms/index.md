Thuộc tính **`platforms`** của giao diện {{domxref("BeforeInstallPromptEvent")}} liệt kê các nền tảng mà sự kiện đã được gửi tới. Điều này được cung cấp cho các user agent muốn đưa ra cho người dùng lựa chọn giữa nhiều phiên bản, chẳng hạn như "web" hoặc "play", để người dùng có thể chọn giữa phiên bản web hoặc phiên bản Android.

## Giá trị

Một mảng chuỗi, trong đó mỗi chuỗi xác định một nền tảng đích cho việc cài đặt.
