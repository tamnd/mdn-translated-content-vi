Thuộc tính **`length`** của giao diện {{domxref("ImageTrackList")}} trả về độ dài của `ImageTrackList`.

## Giá trị

Một số nguyên.

## Ví dụ

Ví dụ sau in giá trị của `length` vào bảng điều khiển.

```js
let tracks = imageDecoder.tracks;
console.log(tracks.length);
```
