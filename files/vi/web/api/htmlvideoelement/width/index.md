{{APIRef("HTML DOM")}}

Thuộc tính **`width`** của {{domxref("HTMLVideoElement")}} trả về một số nguyên phản ánh thuộc tính `width` của phần tử {{HTMLElement("video")}}, chỉ định chiều rộng hiển thị của tài nguyên theo CSS pixel.
