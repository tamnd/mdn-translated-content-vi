{{APIRef("HTML DOM")}}

Thuộc tính chỉ đọc **`videoHeight`** của {{domxref("HTMLVideoElement")}} cho biết chiều cao gốc của video, tính bằng CSS pixel. Nếu chưa có media, giá trị là `0`.
