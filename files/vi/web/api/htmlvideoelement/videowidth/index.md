{{APIRef("HTML DOM")}}

Thuộc tính chỉ đọc **`videoWidth`** của {{domxref("HTMLVideoElement")}} cho biết chiều rộng gốc của video, tính bằng CSS pixel. Nếu chưa có media, giá trị là `0`.
