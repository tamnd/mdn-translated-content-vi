Hàm khởi tạo **`PresentationRequest()`** tạo một đối tượng {{domxref("PresentationRequest")}} mới.

## Cú pháp

```js-nolint
new PresentationRequest(url)
new PresentationRequest(urls)
```

### Tham số

- `url` hoặc `urls\[]`
  - : Một URL hoặc mảng các URL là các URL có thể được sử dụng để tạo hoặc kết nối lại với một bản trình chiếu cho phiên bản PresentationRequest.
