Hàm khởi tạo **`CSSTransformValue()`** tạo một đối tượng {{domxref("CSSTransformValue")}} mới đại diện cho danh sách các đối tượng biến đổi riêng lẻ.

## Cú pháp

```js-nolint
new CSSTransformValue(transforms)
```

### Tham số

- `transforms`
  - : Danh sách các đối tượng {{domxref("CSSTransformComponent")}} để duyệt qua.

### Giá trị trả về

Một đối tượng {{domxref("CSSTransformValue")}} mới.

### Ngoại lệ

- {{jsxref("TypeError")}}
  - : Được ném ra nếu transforms rỗng.
