{{APIRef("HTML DOM")}}

Phương thức **`HTMLSelectElement.setCustomValidity()`** đặt thông báo hợp lệ tùy chỉnh cho phần tử chọn thành thông báo được chỉ định. Dùng chuỗi rỗng để cho biết phần tử _không_ có lỗi hợp lệ tùy chỉnh.

## Cú pháp

```js-nolint
setCustomValidity(string)
```

### Tham số

- `string`
  - : Thông báo hợp lệ tùy chỉnh sẽ được dùng.

### Giá trị trả về

Không có.
