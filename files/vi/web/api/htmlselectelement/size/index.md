{{APIRef("HTML DOM")}}

Thuộc tính **`size`** của giao diện {{DOMxRef("HTMLSelectElement")}} phản ánh thuộc tính HTML [`size`](/en-US/docs/Web/HTML/Reference/Elements/select#size), chứa số lượng mục hiển thị trong điều khiển. Mặc định là 1, trừ khi `multiple` là `true`, khi đó là 4.

## Giá trị

Một số nguyên dài.
