{{APIRef("Media Source Extensions")}}

Thuộc tính chỉ đọc **`sourceBuffer`** của **{{domxref("VideoTrack")}}** trả về {{domxref("SourceBuffer")}} đã tạo ra track, hoặc null nếu track không được tạo bởi {{domxref("SourceBuffer")}} hoặc {{domxref("SourceBuffer")}} đã bị xóa khỏi thuộc tính {{domxref("MediaSource.sourceBuffers")}} của nguồn media cha.

## Giá trị

Một {{domxref("SourceBuffer")}} hoặc null.
