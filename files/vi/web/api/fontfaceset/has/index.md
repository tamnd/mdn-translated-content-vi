{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

Phương thức **`has()`** của giao diện {{domxref("FontFaceSet")}} trả về {{jsxref("Boolean")}} khẳng định xem phần tử có giá trị đã cho có tồn tại không.

## Cú pháp

```js-nolint
has(value)
```

### Tham số

- `value`
  - : Giá trị cần kiểm tra trong đối tượng `FontFaceSet`.

### Giá trị trả về

{{jsxref("Boolean")}}, `true` nếu `value` tồn tại trong `FontFaceSet`.
