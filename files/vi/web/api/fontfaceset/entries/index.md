{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

Phương thức **`entries()`** của giao diện {{domxref("FontFaceSet")}} trả về đối tượng {{jsxref("Iterator")}} mới, chứa mảng `[value,value]` cho mỗi phần tử trong `FontFaceSet`.

## Cú pháp

```js-nolint
entries()
```

### Tham số

Không có.

### Giá trị trả về

Iterator mới chứa mảng `[value, value]` cho mỗi phần tử trong `CustomStateSet`, theo thứ tự chèn.
