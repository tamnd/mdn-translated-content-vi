{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

Thuộc tính **`size`** của giao diện {{domxref("FontFaceSet")}} trả về số lượng phần tử trong `FontFaceSet`.

## Giá trị

Số nguyên cho biết số lượng phần tử trong `FontFaceSet`.

## Ví dụ

Trong ví dụ sau, `size` của `FontFaceSet` được in ra console.

```js
console.log(document.fonts.size);
```
