{{APIRef("CSS Typed Object Model API")}}

Giao diện **`StylePropertyMap`** của [CSS Typed Object Model API](/en-US/docs/Web/API/CSS_Object_Model#css_typed_object_model) cung cấp biểu diễn của một khối khai báo CSS là một thay thế cho {{DOMxRef("CSSStyleDeclaration")}}.

{{InheritanceDiagram}}

## Thuộc tính phiên bản

_Kế thừa thuộc tính từ đối tượng cha, {{DOMxRef("StylePropertyMapReadOnly")}}._

## Phương thức phiên bản

_Kế thừa phương thức từ đối tượng cha, {{DOMxRef("StylePropertyMapReadOnly")}}._

- {{DOMxRef("StylePropertyMap.append()")}}
  - : Thêm một khai báo CSS mới vào `StylePropertyMap` với thuộc tính và giá trị đã cho.
- {{DOMxRef("StylePropertyMap.clear()")}}
  - : Xóa tất cả các khai báo trong `StylePropertyMap`.
- {{DOMxRef("StylePropertyMap.delete()")}}
  - : Xóa khai báo CSS với thuộc tính đã cho.
- {{DOMxRef("StylePropertyMap.set()")}}
  - : Thay đổi khai báo CSS với thuộc tính đã cho.
