{{APIRef("Web Animations")}}

Thuộc tính chỉ đọc **`Animation.pending`** của [Web Animations API](/en-US/docs/Web/API/Web_Animations_API) cho biết liệu hoạt ảnh hiện có đang chờ một thao tác bất đồng bộ như khởi động phát hoặc tạm dừng một hoạt ảnh đang chạy hay không.

## Giá trị

**`true`** nếu hoạt ảnh đang ở trạng thái pending, **`false`** nếu không.
