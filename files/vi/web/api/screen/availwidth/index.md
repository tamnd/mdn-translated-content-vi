{{APIRef("CSSOM view API")}}

Thuộc tính **`Screen.availWidth`** trả về lượng không gian ngang (tính bằng CSS pixels) khả dụng cho cửa sổ.

## Giá trị

Một số.

## Ví dụ

```js
const screenAvailWidth = window.screen.availWidth;

console.log(screenAvailWidth);
```
