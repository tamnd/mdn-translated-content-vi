Thuộc tính **`width`** của giao diện {{domxref("HTMLTableElement")}} là một chuỗi chứa chiều rộng mong muốn của toàn bộ bảng, tính bằng pixel hoặc phần trăm. Nó phản ánh thuộc tính lỗi thời [`width`](/en-US/docs/Web/HTML/Reference/Elements/table#width).
