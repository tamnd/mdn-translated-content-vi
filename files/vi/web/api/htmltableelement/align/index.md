{{APIRef("HTML DOM")}}{{Deprecated_Header}}

Thuộc tính **`HTMLTableElement.align`** đại diện cho căn chỉnh của bảng.

## Giá trị

Một trong các chuỗi sau:

- `left`
- `center`
- `right`

## Ví dụ

```js
// Set the alignment of a table
const t = document.getElementById("TableA");
t.align = "center";
```
