Giao diện **`CSSMathMax`** của [CSS Typed Object Model API](/en-US/docs/Web/API/CSS_Object_Model) biểu thị hàm CSS {{CSSXref('max','max()')}}. Nó kế thừa thuộc tính và phương thức từ lớp cha {{domxref('CSSNumericValue')}}.

## Hàm khởi tạo

- {{domxref("CSSMathMax.CSSMathMax", "CSSMathMax()")}} {{Experimental_Inline}}
  - : Tạo một đối tượng `CSSMathMax` mới.

## Thuộc tính phiên bản

- {{domxref('CSSMathMax.values')}} {{ReadOnlyInline}}
  - : Trả về một đối tượng {{domxref('CSSNumericArray')}} chứa một hoặc nhiều đối tượng {{domxref('CSSNumericValue')}}.
