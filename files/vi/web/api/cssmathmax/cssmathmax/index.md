Hàm khởi tạo **`CSSMathMax()`** tạo một đối tượng {{domxref("CSSMathMax")}} mới biểu thị hàm CSS {{CSSXref('max', 'max()')}}.

## Cú pháp

```js-nolint
new CSSMathMax(args)
```

### Tham số

- `args`
  - : Danh sách các giá trị cho đối tượng {{domxref('CSSMathProduct')}} là số nguyên double hoặc {{domxref('CSSNumericValue')}}.

### Ngoại lệ

- [`TypeError`](/en-US/docs/Web/JavaScript/Reference/Global_Objects/TypeError)
  - : Được ném ra nếu có _lỗi_ khi cộng tất cả các giá trị trong args.
