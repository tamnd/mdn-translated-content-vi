{{ApiRef("HTML DOM")}}

Thuộc tính chỉ đọc **`type`** của giao diện {{domxref("HTMLTextAreaElement")}} trả về chuỗi `"textarea"`.

## Giá trị

Chuỗi `"textarea"`.

## Ví dụ

```js
const textArea = document.querySelector("textarea");
console.log(textArea.type); // "textarea"
```
