{{ APIRef("HTML DOM") }}

Thuộc tính **`placeholder`** của giao diện {{DOMxRef("HTMLTextAreaElement")}} đại diện cho một gợi ý cho người dùng về nội dung có thể nhập vào điều khiển. Nó phản ánh thuộc tính [`placeholder`](/en-US/docs/Web/HTML/Reference/Elements/textarea#placeholder) của phần tử {{htmlelement("textarea")}}.

## Giá trị

Một chuỗi.

## Ví dụ

```js
const textareaElement = document.getElementById("comment");
console.log(textArea.placeholder);
```
