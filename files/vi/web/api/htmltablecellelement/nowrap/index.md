{{APIRef("HTML DOM API")}}{{deprecated_header}}

Thuộc tính **`noWrap`** của giao diện {{domxref("HTMLTableCellElement")}} trả về một giá trị kiểu logic cho biết liệu văn bản của ô có thể được ngắt thành nhiều dòng hay không.

> [!NOTE]
> Thuộc tính này đã lỗi thời và bạn nên sử dụng thuộc tính CSS {{cssxref("white-space")}} với giá trị `nowrap` thay thế.

## Giá trị

Một giá trị kiểu logic.

## Ví dụ

Thay vào đó, sử dụng CSS `white-space`. Một [ví dụ](/en-US/docs/Web/CSS/Reference/Properties/white-space#controlling_line_wrapping_in_tables) có sẵn trên trang {{cssxref("white-space")}}.
