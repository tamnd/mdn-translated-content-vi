Thuộc tính chỉ đọc **`is2D`** của giao diện {{domxref("CSSTransformComponent")}} cho biết transform là 2D hay 3D.

## Giá trị

Giá trị boolean. True cho biết transform là transform 2D, false nếu là transform 3D.
