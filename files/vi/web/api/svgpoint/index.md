{{APIRef("SVG")}}{{Deprecated_Header}}

**`SVGPoint`** đại diện cho một điểm 2D hoặc 3D trong hệ tọa độ SVG. Nó đã được thay thế trong SVG 2 bằng {{domxref("DOMPoint")}} và {{domxref("DOMPointReadOnly")}}. Tất cả các API từng trả về `SVGPoint`, bao gồm {{domxref("SVGSVGElement.createSVGPoint()")}}, nay trả về {{domxref("DOMPoint")}} thay thế.
