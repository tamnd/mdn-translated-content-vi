Khi viết mã cho Web, bạn có thể dùng một số lượng rất lớn các Web API. Bên dưới là danh sách tất cả API và giao diện (các kiểu đối tượng) mà bạn có thể sử dụng khi phát triển ứng dụng hoặc trang web của mình.

Web API thường được dùng với JavaScript, dù không phải lúc nào cũng bắt buộc như vậy.
