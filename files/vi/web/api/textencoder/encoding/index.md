{{APIRef("Encoding API")}}{{AvailableInWorkers}}

Thuộc tính chỉ đọc **`TextEncoder.encoding`** trả về một chuỗi chứa tên của thuật toán mã hóa mà bộ mã hóa cụ thể này sử dụng.

Nó luôn có giá trị `utf-8`.

## Giá trị

Một chuỗi với giá trị `utf-8`.
