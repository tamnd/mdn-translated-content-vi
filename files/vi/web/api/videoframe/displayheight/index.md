Thuộc tính **`displayHeight`** của giao diện {{domxref("VideoFrame")}} trả về chiều cao của `VideoFrame` sau khi áp dụng điều chỉnh tỷ lệ khung hình.

## Giá trị

Một số nguyên.

## Ví dụ

Ví dụ sau in `displayHeight` ra console.

```js
console.log(VideoFrame.displayHeight);
```
