Thuộc tính **`displayWidth`** của giao diện {{domxref("VideoFrame")}} trả về chiều rộng của `VideoFrame` sau khi áp dụng điều chỉnh tỷ lệ khung hình.

## Giá trị

Một số nguyên.

## Ví dụ

Ví dụ sau in `displayWidth` ra console.

```js
console.log(VideoFrame.displayWidth);
```
