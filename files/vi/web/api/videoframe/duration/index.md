Thuộc tính **`duration`** của giao diện {{domxref("VideoFrame")}} trả về một số nguyên cho biết thời lượng của video tính bằng microsecond.

## Giá trị

Một số nguyên.

## Ví dụ

Ví dụ sau in `duration` ra console.

```js
console.log(VideoFrame.duration);
```
