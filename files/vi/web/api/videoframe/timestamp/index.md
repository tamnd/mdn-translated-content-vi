Thuộc tính **`timestamp`** của giao diện {{domxref("VideoFrame")}} trả về một số nguyên cho biết dấu thời gian của video tính bằng microsecond.

## Giá trị

Một số nguyên.

## Ví dụ

Ví dụ sau in `timestamp` ra console.

```js
console.log(VideoFrame.timestamp);
```
