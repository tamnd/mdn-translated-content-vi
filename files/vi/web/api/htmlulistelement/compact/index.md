Thuộc tính **`compact`** của giao diện {{domxref("HTMLUListElement")}} cho biết khoảng cách giữa các mục danh sách có nên được giảm hay không. Cách xử lý cụ thể thuộc tính `compact` phụ thuộc vào từng trình duyệt. Thay vì sử dụng thuộc tính này, hãy cân nhắc sử dụng CSS {{cssxref("line-height")}}.

Nó phản ánh thuộc tính [`compact`](/en-US/docs/Web/HTML/Reference/Elements/ul#compact) của phần tử {{HTMLElement("ul")}}.

## Giá trị

Một giá trị kiểu logic (boolean).
