Phương thức `createSVGTransform()` của giao diện {{domxref("SVGSVGElement")}} tạo một đối tượng {{domxref("SVGTransform")}} bên ngoài bất kỳ cây tài liệu nào.

## Cú pháp

```js-nolint
createSVGTransform()
```

### Tham số

Không có.

### Giá trị trả về

Một đối tượng {{domxref("SVGTransform")}}, được khởi tạo thành phép biến đổi ma trận đơn vị (`matrix(1, 0, 0, 1, 0, 0)`).
