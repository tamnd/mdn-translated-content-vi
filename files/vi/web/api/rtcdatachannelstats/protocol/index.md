Thuộc tính **`protocol`** của từ điển {{domxref("RTCDataChannelStats")}} trả về chuỗi chứa {{domxref("RTCDataChannel.protocol", "giao thức")}} của kênh dữ liệu liên kết.

Giá trị được xác định bởi trang web hoặc ứng dụng khi tạo kênh dữ liệu.

## Giá trị

Chuỗi chứa cùng giá trị với thuộc tính {{domxref("RTCDataChannel.protocol")}} của kênh dữ liệu liên kết.

Nếu không có giao thức nào được xác định, đây sẽ là chuỗi rỗng ("").
