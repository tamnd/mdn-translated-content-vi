Phương thức tĩnh **`console.groupEnd()`** thoát khỏi nhóm nội tuyến hiện tại trong console. Xem [Sử dụng nhóm trong console](/en-US/docs/Web/API/console#using_groups_in_the_console) trong tài liệu của {{domxref("console")}} để biết chi tiết và ví dụ.

## Cú pháp

```js-nolint
console.groupEnd()
```
