{{APIRef("CSS Painting API")}}

Giao diện **`PaintSize`** của [CSS Painting API](/en-US/docs/Web/API/CSS_Painting_API) đại diện cho kích thước của bitmap đầu ra mà tác giả nên vẽ.

## Thuộc tính phiên bản

- {{domxref('PaintSize.height')}} {{ReadOnlyInline}}
  - : Trả về chiều cao của bitmap đầu ra mà tác giả nên vẽ.
- {{domxref('PaintSize.width')}} {{ReadOnlyInline}}
  - : Trả về chiều rộng của bitmap đầu ra mà tác giả nên vẽ.
