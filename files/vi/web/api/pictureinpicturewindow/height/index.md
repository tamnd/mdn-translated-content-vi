{{APIRef("Picture-in-Picture API")}}

Thuộc tính chỉ đọc **`height`** của giao diện {{domxref("PictureInPictureWindow")}} trả về chiều cao của cửa sổ video nổi tính bằng pixel.

## Giá trị

Giá trị số nguyên cho biết chiều cao của cửa sổ video nổi tính bằng pixel nếu cửa sổ Picture-in-Picture đang mở. Nếu không, nó trả về `0`.
