Phương thức **`createTexture()`** của giao diện {{domxref("GPUDevice")}} tạo một {{domxref("GPUTexture")}} để lưu dữ liệu texture dùng trong các thao tác dựng hình GPU.

## Cú pháp

```js-nolint
createTexture(descriptor)
```

### Tham số

- `descriptor`
  - : Đối tượng mô tả texture cần tạo.

### Giá trị trả về

Một thể hiện {{domxref("GPUTexture")}}.
