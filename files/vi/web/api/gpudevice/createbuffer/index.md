Phương thức **`createBuffer()`** của giao diện {{domxref("GPUDevice")}} tạo một {{domxref("GPUBuffer")}} để lưu dữ liệu thô dùng trong các thao tác GPU.

## Cú pháp

```js-nolint
createBuffer(descriptor)
```

### Tham số

- `descriptor`
  - : Đối tượng mô tả buffer cần tạo.

### Giá trị trả về

Một thể hiện {{domxref("GPUBuffer")}}.
