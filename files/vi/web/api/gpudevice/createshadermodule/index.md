Phương thức **`createShaderModule()`** của giao diện {{domxref("GPUDevice")}} tạo một {{domxref("GPUShaderModule")}} từ một chuỗi mã nguồn WGSL.

## Cú pháp

```js-nolint
createShaderModule(descriptor)
```

### Tham số

- `descriptor`
  - : Đối tượng mô tả module shader cần tạo.

### Giá trị trả về

Một thể hiện {{domxref("GPUShaderModule")}}.
