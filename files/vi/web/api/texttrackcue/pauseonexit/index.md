Thuộc tính **`pauseOnExit`** của giao diện {{domxref("TextTrackCue")}} trả về hoặc đặt cờ cho biết liệu việc phát phương tiện có nên tạm dừng khi đến cuối phạm vi mà cue này áp dụng hay không.

## Giá trị

Một {{jsxref("Boolean")}}, true nếu phương tiện sẽ tạm dừng.
