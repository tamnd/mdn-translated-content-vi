Thuộc tính chỉ đọc **`type`** của giao diện {{domxref("Credential")}} trả về một chuỗi chứa kiểu của thông tin xác thực. Các giá trị hợp lệ là `password`, `federated`, `public-key`, `identity` và `otp`.

## Giá trị

Một chuỗi chứa kiểu của thông tin xác thực.
