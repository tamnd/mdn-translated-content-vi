Phương thức **`CSSUnparsedValue.values()`** trả về một đối tượng _array iterator_ mới chứa các giá trị cho mỗi chỉ mục trong đối tượng CSSUnparsedValue.

## Cú pháp

```js-nolint
values()
```

### Tham số

Không có.

### Giá trị trả về

Một {{jsxref("Array")}} mới.
