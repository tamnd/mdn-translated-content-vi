Phương thức **`CSSUnparsedValue.keys()`** trả về một đối tượng _array iterator_ mới chứa các khóa cho mỗi chỉ mục trong mảng.

## Cú pháp

```js-nolint
keys()
```

### Tham số

Không có.

### Giá trị trả về

Một {{jsxref("Array")}} mới.
