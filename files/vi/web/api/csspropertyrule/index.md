{{APIRef("CSS Properties and Values API")}}

Giao diện **`CSSPropertyRule`** của [CSS Properties and Values API](/en-US/docs/Web/API/CSS_Properties_and_Values_API) đại diện cho quy tắc CSS `@property` duy nhất.

{{InheritanceDiagram}}

## Thuộc tính phiên bản

_Kế thừa các thuộc tính từ tổ tiên của nó {{domxref("CSSRule")}}._

- {{domxref("CSSPropertyRule.inherits")}} {{ReadOnlyInline}}
  - : Trả về cờ kế thừa của quy tắc thuộc tính tùy chỉnh.
- {{domxref("CSSPropertyRule.initialValue")}} {{ReadOnlyInline}}
  - : Trả về giá trị ban đầu của quy tắc thuộc tính tùy chỉnh.
- {{domxref("CSSPropertyRule.name")}} {{ReadOnlyInline}}
  - : Trả về tên của quy tắc thuộc tính tùy chỉnh, tức là tên của thuộc tính.
- {{domxref("CSSPropertyRule.syntax")}} {{ReadOnlyInline}}
  - : Trả về cú pháp theo nghĩa đen của quy tắc thuộc tính tùy chỉnh.

## Phương thức phiên bản

_Không có phương thức nào cụ thể; kế thừa các phương thức từ tổ tiên của nó {{domxref("CSSRule")}}._
