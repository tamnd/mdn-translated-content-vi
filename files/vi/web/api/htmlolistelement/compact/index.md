{{ApiRef("HTML DOM")}}{{Deprecated_Header}}

Thuộc tính **`compact`** của giao diện {{domxref("HTMLOListElement")}} cho biết rằng khoảng cách giữa các mục danh sách nên được giảm. Cách xử lý chính xác thuộc tính `compact` phụ thuộc vào trình duyệt. Thay vì sử dụng thuộc tính này, hãy xem xét sử dụng CSS {{cssxref("line-height")}}.

Nó phản ánh thuộc tính [`compact`](/en-US/docs/Web/HTML/Reference/Elements/ol#compact) của phần tử {{HTMLElement("ol")}}.

## Giá trị

Một giá trị boolean.
