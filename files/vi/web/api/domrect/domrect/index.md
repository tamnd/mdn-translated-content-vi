Hàm khởi tạo **`DOMRect()`** tạo một đối tượng {{domxref("DOMRect")}} mới.

## Cú pháp

```js-nolint
new DOMRect(x, y, width, height)
```

### Tham số

- {{domxref("DOMRect.x", "x")}}
  - : Tọa độ `x` của gốc `DOMRect`.
- {{domxref("DOMRect.y", "y")}}
  - : Tọa độ `y` của gốc `DOMRect`.
- {{domxref("DOMRect.width", "width")}}
  - : Chiều rộng của `DOMRect`.
- {{domxref("DOMRect.height", "height")}}
  - : Chiều cao của `DOMRect`.

### Giá trị trả về

Một đối tượng {{domxref("DOMRect")}} mới.
