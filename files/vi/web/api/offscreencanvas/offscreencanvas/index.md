{{APIRef("Canvas API")}}

Hàm khởi tạo **`OffscreenCanvas()`** tạo một đối tượng {{domxref("OffscreenCanvas")}} mới.

## Cú pháp

```js-nolint
new OffscreenCanvas(width, height)
```

### Tham số

- `width`
  - : Chiều rộng của `OffscreenCanvas`.
- `height`
  - : Chiều cao của `OffscreenCanvas`.

## Ví dụ

```js
const offscreen = new OffscreenCanvas(256, 256);
```
