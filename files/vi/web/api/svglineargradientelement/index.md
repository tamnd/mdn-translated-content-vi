  - : Một {{domxref("SVGAnimatedLength")}} tương ứng với thuộc tính {{SVGAttr("y1")}} của phần tử {{SVGElement("linearGradient")}} đã cho.
- {{domxref("SVGLinearGradientElement.x2")}} {{ReadOnlyInline}}
  - : Một {{domxref("SVGAnimatedLength")}} tương ứng với thuộc tính {{SVGAttr("x2")}} của phần tử {{SVGElement("linearGradient")}} đã cho.
