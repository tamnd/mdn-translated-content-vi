{{APIRef("HTML DOM")}}

Phương thức này luôn trả về sai.

## Cú pháp

```js-nolint
javaEnabled()
```

### Thông số

Không có.

### Giá trị trả về

Giá trị boolean `false`.

## Ví dụ

```js
if (window.navigator.javaEnabled()) {
  // code will never be executed; the condition is always false
}
```
