{{APIRef("HTML DOM")}}

Phương thức **`Navigator.taintEnabled()`** luôn trả về
`false`.

Làm hỏng là một phương pháp bảo mật được sử dụng bởi JavaScript 1.2. Nó đã bị dỡ bỏ từ lâu; cái này
phương pháp này chỉ duy trì khả năng tương thích với các tập lệnh rất cũ.

## Cú pháp

```js-nolint
taintEnabled()
```

### Thông số

Không có.

### Giá trị trả về

Luôn trả về `false`.
