{{APIRef("HTML DOM")}}

Thuộc tính chỉ đọc **`maxTouchPoints`** của
Giao diện {{domxref("Navigator")}} trả về số lần chạm đồng thời tối đa
các điểm liên lạc được thiết bị hiện tại hỗ trợ.

## Giá trị

Một con số.

## Ví dụ

```js
if (navigator.maxTouchPoints > 1) {
  // browser supports multi-touch
}
```
