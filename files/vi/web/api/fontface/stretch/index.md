{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

Thuộc tính **`stretch`** của giao diện {{domxref("FontFace")}} lấy hoặc đặt cách font co giãn.

Thuộc tính này tương đương với descriptor {{cssxref("@font-face/font-stretch", "font-stretch")}}.

## Giá trị

Một chuỗi chứa descriptor như nó xuất hiện trong quy tắc `@font-face` của style sheet.
