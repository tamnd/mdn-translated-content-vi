{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

Thuộc tính **`weight`** của giao diện {{domxref("FontFace")}} lấy hoặc đặt độ đậm của font.

Thuộc tính này tương đương với descriptor {{cssxref("@font-face/font-weight", "font-weight")}}.

## Giá trị

Một chuỗi chứa descriptor như nó được định nghĩa trong quy tắc `@font-face` của style sheet.
