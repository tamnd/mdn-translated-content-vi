{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

Thuộc tính **`unicodeRange`** của giao diện {{domxref("FontFace")}} lấy hoặc đặt phạm vi code point unicode bao gồm trong font.

Thuộc tính này tương đương với descriptor {{cssxref("@font-face/unicode-range", "unicode-range")}}.

## Giá trị

Một chuỗi chứa descriptor như nó xuất hiện trong quy tắc `@font-face` của style sheet.
