{{APIRef("CSS Font Loading API")}}{{AvailableInWorkers}}

Thuộc tính **`style`** của giao diện {{domxref("FontFace")}} lấy hoặc đặt kiểu của font.

Thuộc tính này tương đương với descriptor {{cssxref("@font-face/font-style", "font-style")}}.

## Giá trị

Một chuỗi chứa các descriptor được định nghĩa trong quy tắc `@font-face` của style sheet.
