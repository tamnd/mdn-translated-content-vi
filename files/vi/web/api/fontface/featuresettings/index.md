Thuộc tính **`featureSettings`** của giao diện {{domxref("FontFace")}} lấy hoặc đặt các tính năng font ít dùng không có sẵn từ các thuộc tính variant của font.

Thuộc tính này tương đương với descriptor {{cssxref("font-feature-settings")}}.

## Giá trị

Một chuỗi chứa descriptor.
