{{APIRef("HTML DOM")}}

Phương thức **`setCustomValidity()`** của giao diện {{domxref("HTMLButtonElement")}} đặt thông báo hợp lệ tùy chỉnh cho phần tử. Dùng chuỗi rỗng để cho biết phần tử _không_ có lỗi hợp lệ tùy chỉnh.

## Cú pháp

```js
setCustomValidity(string);
```
