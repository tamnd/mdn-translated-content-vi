{{APIRef("HTML DOM")}}

Phương thức **`HTMLButtonElement.checkValidity()`** trả về `true` nếu giá trị của phần tử không có vấn đề về tính hợp lệ; ngược lại, nó trả về `false`.

## Cú pháp

```js
checkValidity();
```
