{{APIRef("HTML DOM")}}

Thuộc tính **`popoverTargetElement`** của giao diện {{domxref("HTMLButtonElement")}} lấy và đặt phần tử popover được điều khiển qua nút. Đây là bản tương đương JavaScript của thuộc tính HTML [`popovertarget`](/en-US/docs/Web/HTML/Reference/Elements/button#popovertarget).

## Giá trị

Tham chiếu đến một phần tử popover.
