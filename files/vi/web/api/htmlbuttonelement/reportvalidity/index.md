{{APIRef("HTML DOM")}}

Phương thức **`reportValidity()`** của giao diện {{domxref("HTMLButtonElement")}} thực hiện cùng hành động như `checkValidity()`, nhưng cũng báo cáo kết quả cho người dùng nếu sự kiện `invalid` không bị hủy.

## Cú pháp

```js
reportValidity();
```

## Giá trị trả về

Một giá trị boolean.
