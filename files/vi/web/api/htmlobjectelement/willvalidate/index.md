{{APIRef("HTML DOM")}}

Thuộc tính chỉ đọc **`willValidate`** của giao diện {{domxref("HTMLObjectElement")}} trả về `false`, vì các phần tử {{HTMLElement("object")}} không phải là ứng cử viên cho [xác thực ràng buộc](/en-US/docs/Web/HTML/Guides/Constraint_validation).

## Giá trị

Giá trị boolean `false`.
