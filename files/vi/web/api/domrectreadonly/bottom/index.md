Thuộc tính **`bottom`** chỉ đọc của giao diện **`DOMRectReadOnly`** trả về giá trị tọa độ dưới cùng của `DOMRect`. (Có cùng giá trị với `y + height`, hoặc `y` nếu `height` âm.)

## Giá trị

Một số thực dấu phẩy động.
