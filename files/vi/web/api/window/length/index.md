{{APIRef}}

Returns the number of frames (either {{HTMLElement("frame")}} or
{{HTMLElement("iframe")}} elements) in the window.

## Giá trị

A number.

## Ví dụ

```js
if (window.length) {
  // this is a document with subframes
}
```
