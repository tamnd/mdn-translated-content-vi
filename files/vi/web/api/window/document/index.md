{{APIRef}}

**`window.document`** returns a reference to the [document](/en-US/docs/Web/API/Document) contained in the window.

## Giá trị

A [document](/en-US/docs/Web/API/Document) object.

## Ví dụ

```js
console.log(window.document.title);
```
