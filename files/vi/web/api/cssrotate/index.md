{{APIRef("CSS Typed Object Model API")}}

Giao diện **`CSSRotate`** của [CSS Typed Object Model API](/en-US/docs/Web/API/CSS_Object_Model) đại diện cho giá trị rotate của thuộc tính {{CSSXRef('transform')}} riêng lẻ trong CSS. Nó kế thừa các thuộc tính và phương thức từ lớp cha {{domxref('CSSTransformValue')}}.

{{InheritanceDiagram}}

## Hàm khởi tạo

- {{domxref("CSSRotate.CSSRotate", "CSSRotate()")}}
  - : Tạo một đối tượng `CSSRotate` mới.

## Thuộc tính phiên bản

- {{domxref('CSSRotate.x','x')}}
  - : Trả về hoặc đặt giá trị trục x.
- {{domxref('CSSRotate.y','y')}}
  - : Trả về hoặc đặt giá trị trục y.
- {{domxref('CSSRotate.z','z')}}
  - : Trả về hoặc đặt giá trị trục z.
- {{domxref('CSSRotate.angle','angle')}}
  - : Trả về hoặc đặt giá trị góc.
