Thuộc tính chỉ đọc **`type`** của giao diện {{domxref("SVGComponentTransferFunctionElement")}} phản ánh thuộc tính {{SVGAttr("type")}} của phần tử đã cho. Nó nhận một trong các hằng số [`SVG_FECOMPONENTTRANSFER_TYPE_*`](/en-US/docs/Web/API/SVGComponentTransferFunctionElement#static_properties) được định nghĩa trên giao diện này.
