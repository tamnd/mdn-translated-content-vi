Giao diện **`DeviceMotionEventRotationRate`** của {{domxref("Device Orientation Events", "", "", "nocode")}} cung cấp thông tin về tốc độ quay của thiết bị xung quanh cả ba trục.

## Thuộc tính phiên bản

- {{ domxref("DeviceMotionEventRotationRate.alpha") }} {{ReadOnlyInline}}
  - : Lượng quay xung quanh trục Z, tính bằng độ mỗi giây.
- {{ domxref("DeviceMotionEventRotationRate.beta") }} {{ReadOnlyInline}}
  - : Lượng quay xung quanh trục X, tính bằng độ mỗi giây.
- {{ domxref("DeviceMotionEventRotationRate.gamma") }} {{ReadOnlyInline}}
  - : Lượng quay xung quanh trục Y, tính bằng độ mỗi giây.
