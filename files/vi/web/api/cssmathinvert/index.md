{{APIRef("CSS Typed Object Model API")}}

Giao diện **`CSSMathInvert`** của [CSS Typed Object Model API](/en-US/docs/Web/API/CSS_Object_Model) biểu thị một CSS {{CSSXref('calc','calc()')}} được sử dụng dưới dạng `calc(1 / <value>)`. Nó kế thừa thuộc tính và phương thức từ lớp cha {{domxref('CSSNumericValue')}}.

{{InheritanceDiagram}}

## Hàm khởi tạo

- {{domxref("CSSMathInvert.CSSMathInvert", "CSSMathInvert()")}}
  - : Tạo một đối tượng `CSSMathInvert` mới.

## Thuộc tính phiên bản

- {{domxref('CSSMathInvert.value')}} {{ReadOnlyInline}}
  - : Trả về một đối tượng {{domxref('CSSNumericValue')}}.

## Phương thức tĩnh

_Giao diện cũng có thể kế thừa phương thức từ giao diện cha {{domxref("CSSMathValue")}}._
