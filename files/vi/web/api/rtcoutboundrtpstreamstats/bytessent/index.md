{{APIRef("WebRTC")}}

Thuộc tính **`bytesSent`** của từ điển {{domxref("RTCOutboundRtpStreamStats")}} cho biết tổng số byte dữ liệu đã gửi trên luồng này ({{domxref("RTCDataChannel")}}).

Giá trị bao gồm cả byte dữ liệu được truyền lại. Không bao gồm header hay phần đệm.

## Giá trị

Một số nguyên dương.
