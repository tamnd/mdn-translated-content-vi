{{APIRef("WebRTC")}}

Thuộc tính **`mediaSourceId`** của từ điển {{domxref("RTCOutboundRtpStreamStats")}} biểu thị ID của đối tượng thống kê của track hiện đang gắn với bộ gửi của luồng này.

ID này sau đó có thể được dùng để lấy {{domxref("RTCVideoSourceStats")}} hoặc {{domxref("RTCAudioSourceStats")}} tương ứng.

## Giá trị

Chuỗi chứa ID của đối tượng thống kê nguồn phương tiện.
