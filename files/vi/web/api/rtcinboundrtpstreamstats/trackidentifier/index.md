Thuộc tính **`trackIdentifier`** của từ điển {{domxref("RTCInboundRtpStreamStats")}} là một chuỗi xác định `MediaStreamTrack` liên kết với luồng đến.

Giá trị này sẽ khớp với giá trị {{domxref("MediaStreamTrack.id")}} của track liên kết.

## Giá trị

Một chuỗi xác định track media liên kết.
