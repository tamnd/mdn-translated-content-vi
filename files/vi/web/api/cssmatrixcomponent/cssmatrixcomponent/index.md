Hàm khởi tạo **`CSSMatrixComponent()`** tạo một đối tượng {{domxref("CSSMatrixComponent")}} mới biểu thị các giá trị {{cssxref("transform-function/matrix", "matrix()")}} và {{cssxref("transform-function/matrix3d", "matrix3d()")}} của thuộc tính {{CSSXRef('transform')}} riêng lẻ trong CSS.

## Cú pháp

```js-nolint
new CSSMatrixComponent(matrix)
new CSSMatrixComponent(matrix, options)
```

### Tham số

- {{domxref('CSSMatrixComponent.matrix','matrix')}}
  - : Một ma trận 2d hoặc 3d.
