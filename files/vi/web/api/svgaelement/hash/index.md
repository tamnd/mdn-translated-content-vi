{{APIRef("SVG")}}

Thuộc tính **`hash`** của giao diện {{domxref("SVGAElement")}} trả về một chuỗi biểu thị mã định danh mảnh, bao gồm dấu thăng (`#`) đứng đầu nếu có, trong URL được tham chiếu.

Thuộc tính này có thể được đặt để thay đổi mảnh của URL (bỏ qua dấu `#` đứng đầu).

## Giá trị

Một chuỗi.
