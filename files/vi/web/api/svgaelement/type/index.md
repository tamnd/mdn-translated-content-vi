{{APIRef("SVG")}}

Thuộc tính **`type`** của giao diện {{domxref("SVGAElement")}} trả về một chuỗi cho biết kiểu MIME của tài nguyên được liên kết.

Thuộc tính này có thể được đặt để thay đổi giá trị `type` của URL. Nó phản ánh giá trị của thuộc tính [`type`](/en-US/docs/Web/HTML/Reference/Elements/a#type).

## Giá trị

Một chuỗi.
