{{APIRef("HTML DOM")}}

Thuộc tính **`hash`** của giao diện {{DOMxRef("HTMLAreaElement")}} là một chuỗi chứa định danh phân đoạn của URL được tham chiếu, nếu có, bao gồm dấu thăng `#` ở đầu.

Nó phản ánh thuộc tính [`hash`](/en-US/docs/Web/HTML/Reference/Elements/area#hash) của phần tử {{htmlelement("area")}}.

## Giá trị

Một chuỗi.

## Ví dụ

```js
const area = document.getElementById("imageArea");
console.log(area.hash);
```
