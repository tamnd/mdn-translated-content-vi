{{APIRef("HTML DOM")}}

Thuộc tính **`hostname`** của giao diện {{DOMxRef("HTMLAreaElement")}} là một chuỗi chứa tên máy chủ của URL được tham chiếu.

Nó phản ánh thuộc tính [`hostname`](/en-US/docs/Web/HTML/Reference/Elements/area#hostname) của phần tử {{htmlelement("area")}}.

## Giá trị

Một chuỗi.

## Ví dụ

```js
const area = document.getElementById("imageArea");
console.log(area.hostname);
```
