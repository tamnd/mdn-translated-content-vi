{{APIRef("HTML DOM")}}

Thuộc tính **`host`** của giao diện {{DOMxRef("HTMLAreaElement")}} là một chuỗi chứa tên máy chủ và cổng, nếu không phải là cổng mặc định, của URL được tham chiếu.

Nó phản ánh thuộc tính [`host`](/en-US/docs/Web/HTML/Reference/Elements/area#host) của phần tử {{htmlelement("area")}}.

## Giá trị

Một chuỗi.

## Ví dụ

```js
const area = document.getElementById("imageArea");
console.log(area.host);
```
