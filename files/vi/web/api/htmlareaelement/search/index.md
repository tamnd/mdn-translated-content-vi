{{APIRef("HTML DOM")}}

Thuộc tính **`search`** của giao diện {{DOMxRef("HTMLAreaElement")}} là một chuỗi chứa thành phần truy vấn, nếu có, của URL được tham chiếu, bao gồm dấu hỏi `?` ở đầu.

Nó phản ánh thuộc tính [`search`](/en-US/docs/Web/HTML/Reference/Elements/area#search) của phần tử {{htmlelement("area")}}.

## Giá trị

Một chuỗi.
