{{APIRef("HTML DOM")}}

Thuộc tính **`HTMLMediaElement.paused`** chỉ đọc
cho biết phần tử media có bị tạm dừng hay không.

## Giá trị

Một giá trị boolean. `true` bị tạm dừng và `false` thì không
tạm dừng.

## Ví dụ

```js
const obj = document.createElement("video");
console.log(obj.paused); // true
```
