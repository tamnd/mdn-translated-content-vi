{{APIRef("HTML DOM")}}

Thuộc tính **`HTMLMediaElement.loop`** phản ánh thuộc tính HTML [`loop`](/en-US/docs/Web/HTML/Reference/Elements/video#loop), thuộc tính này kiểm soát xem phần tử phương tiện có nên bắt đầu lại khi nó kết thúc hay không.

## Giá trị

Một giá trị boolean.

## Ví dụ

```js
const obj = document.createElement("video");
obj.loop = true; // true
```
