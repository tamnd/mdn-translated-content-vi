{{APIRef("DOM")}}

Thuộc tính chỉ đọc **`ownerDocument`** của giao diện {{domxref("Node")}} trả về {{domxref("Document")}} mà nút này thuộc về. Nếu chính nút là một tài liệu, thuộc tính trả về `null`.

## Giá trị

Một {{domxref("Document")}} mà nút hiện tại thuộc về, hoặc `null` nếu nút là tài liệu.
