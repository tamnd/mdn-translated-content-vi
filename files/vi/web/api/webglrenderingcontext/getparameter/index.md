Phương thức **`WebGLRenderingContext.getParameter()`** của [WebGL API](/en-US/docs/Web/API/WebGL_API) trả về một giá trị cho tên tham số đã truyền.

## Cú pháp

```js-nolint
getParameter(pname)
```

### Thông số

- `pname`
  - : Một {{domxref("WebGL_API/Types", "GLenum")}} chỉ định giá trị tham số nào cần trả về. Xem bên dưới để biết
những giá trị có thể.

### Giá trị trả về

Phụ thuộc vào tham số.
