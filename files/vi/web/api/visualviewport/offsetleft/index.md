Thuộc tính **`offsetLeft`** chỉ đọc của giao diện {{domxref("VisualViewport")}} trả về độ lệch của cạnh trái khung nhìn trực quan so với cạnh trái khung nhìn bố cục tính bằng pixel CSS, hoặc `0` nếu tài liệu hiện tại không hoàn toàn hoạt động.

## Giá trị

Một số thực dấu phẩy động.
