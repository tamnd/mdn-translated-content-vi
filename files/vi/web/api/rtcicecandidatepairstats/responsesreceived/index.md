Thuộc tính **`responsesReceived`** của từ điển {{domxref("RTCIceCandidatePairStats")}} cho biết tổng số phản hồi kiểm tra kết nối {{Glossary("STUN")}} đã nhận trên kết nối được mô tả bởi cặp ứng viên này.

## Giá trị
