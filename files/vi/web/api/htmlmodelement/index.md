{{ APIRef("HTML DOM") }}

Giao diện **`HTMLModElement`** cung cấp các thuộc tính đặc biệt (beyond các phương thức và thuộc tính thông thường có sẵn thông qua giao diện {{domxref("HTMLElement")}} mà chúng cũng kế thừa) để thao tác các phần tử sửa đổi, cụ thể là {{HTMLElement("del")}} và {{HTMLElement("ins")}}.

{{InheritanceDiagram}}

## Thuộc tính instance

_Kế thừa các thuộc tính từ phần tử cha, {{domxref("HTMLElement")}}._

- {{domxref("HTMLModElement.cite")}}
  - : Một chuỗi phản ánh thuộc tính HTML [`cite`](/en-US/docs/Web/HTML/Reference/Elements/del#cite), chứa URI của tài nguyên giải thích sự thay đổi.
- {{domxref("HTMLModElement.dateTime")}}
  - : Một chuỗi phản ánh thuộc tính HTML [`datetime`](/en-US/docs/Web/HTML/Reference/Elements/del#datetime), chứa một chuỗi ngày-giờ đại diện cho mốc thời gian của sự thay đổi.
