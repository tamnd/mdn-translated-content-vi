Thuộc tính chỉ đọc **`type`** của giao diện {{domxref("NetworkInformation")}} trả về loại kết nối mà thiết bị đang sử dụng để liên lạc với mạng.

## Giá trị

Một giá trị kiểu liệt kê, là một trong các giá trị sau:

- `"bluetooth"`
- `"cellular"`
- `"ethernet"`
- `"none"`
- `"wifi"`
- `"wimax"`
- `"other"`
- `"unknown"`
