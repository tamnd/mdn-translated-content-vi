**`<ms>`** [MathML](/en-US/docs/Web/MathML) element đại diện cho một hằng **chuỗi** được thiết kế để các ngôn ngữ lập trình và hệ đại số máy tính diễn giải.

## Thuộc tính

Các thuộc tính của phần tử này bao gồm [thuộc tính MathML toàn cục](/en-US/docs/Web/MathML/Reference/Global_attributes).

## Ví dụ

### Hiển thị mặc định

```html
<math display="block">
  <ms>Hello World!</ms>
</math>
```

### Thuộc tính dấu ngoặc kiểu cũ

```html
<math display="block">
  <ms lquote="„" rquote="'">abc</ms>
</math>
```
