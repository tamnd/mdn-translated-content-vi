**`<mroot>`** [MathML](/en-US/docs/Web/MathML) element được dùng để hiển thị căn với chỉ số căn rõ ràng. Nó nhận hai đối số, dẫn đến cú pháp: `<mroot> base index </mroot>`.

## Thuộc tính

Phần tử này chấp nhận các [thuộc tính MathML toàn cục](/en-US/docs/Web/MathML/Reference/Global_attributes).

## Ví dụ

```html
<math display="block">
  <mroot>
    <mi>x</mi>
    <mn>3</mn>
  </mroot>
</math>
```
