Phần tử `<xsl:call-template>` gọi một template đã đặt tên.

## Cú pháp

```xml
<xsl:call-template name=NAME>
  <xsl:with-param> [optional]
</xsl:call-template>
```

### Thuộc tính bắt buộc

- `name`
  - : Chỉ định tên của template mà bạn muốn gọi.

### Thuộc tính tùy chọn

Không có.

### Kiểu

Chỉ thị, xuất hiện bên trong một template.
