`set:has-same-node()` xác định liệu hai node-set có bất kỳ node nào chung hay không.

## Cú pháp

```plain
set:has-same-node(nodeSet1, nodeSet2)
```

### Tham số

- `nodeSet1`
  - : Node-set thứ nhất cần kiểm tra.
- `nodeSet2`
  - : Node-set thứ hai cần kiểm tra.

### Giá trị trả về

`true` nếu hai node-set có bất kỳ node nào chung; ngược lại là `false`.
