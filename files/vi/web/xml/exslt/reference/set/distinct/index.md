`set:distinct()` trả về một tập con các node trong node-set được chỉ định, chỉ trả về các node có giá trị chuỗi duy nhất.

## Cú pháp

```plain
set:distinct(nodeSet)
```

### Tham số

- `nodeSet`
  - : Node-set trong đó cần tìm các node duy nhất.

### Giá trị trả về

Một node-set chứa các node có giá trị chuỗi duy nhất.
