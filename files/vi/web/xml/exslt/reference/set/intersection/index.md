`set:intersection()` trả về phần giao của hai node-set. Nói cách khác, nó trả về một node-set chứa tất cả các node tồn tại trong cả `nodeSet1` và `nodeSet2`.

## Cú pháp

```plain
set:intersection(nodeSet1, nodeSet2)
```

### Tham số

- `nodeSet1`
  - : Node-set thứ nhất.
- `nodeSet2`
  - : Node-set thứ hai.

### Giá trị trả về

Một node-set chứa các node tồn tại trong cả `nodeSet1` và `nodeSet2`.
