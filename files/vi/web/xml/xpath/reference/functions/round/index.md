Hàm `round` trả về một số là số nguyên gần nhất với số đã cho.

## Cú pháp

```plain
round( decimal )
```

### Tham số

- `decimal`
  - : Số thập phân cần làm tròn.

### Giá trị trả về

Số nguyên gần nhất nhỏ hơn, lớn hơn hoặc bằng `decimal`.

## Mô tả

- -0.5 làm tròn thành số âm không. 0.4 làm tròn thành số dương không.
