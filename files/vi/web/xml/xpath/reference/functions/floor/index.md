Hàm `floor` đánh giá một số thập phân và trả về số nguyên lớn nhất nhỏ hơn hoặc bằng số thập phân đó.

## Cú pháp

```plain
floor( number )
```

### Tham số

- `number`
  - : Số thập phân cần đánh giá.

### Giá trị trả về

Số nguyên gần nhất nhỏ hơn hoặc bằng `number`.
