Hàm `function-available` xác định xem một hàm cho trước có sẵn hay không và trả về boolean true hoặc false.

## Cú pháp

```plain
function-available( name )
```

### Tham số

- `name`
  - : Tên của hàm cần kiểm tra.

### Giá trị trả về

Boolean `true` hoặc `false`.
