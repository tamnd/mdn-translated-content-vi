Hàm `contains` xác định liệu chuỗi đối số thứ nhất có chứa chuỗi đối số thứ hai hay không và trả về boolean true hoặc false.

## Cú pháp

```plain
contains(haystack, needle)
```

### Tham số

- `haystack`
  - : Chuỗi sẽ được tìm kiếm
- `needle`
  - : Chuỗi cần tìm như một chuỗi con của `haystack`

### Giá trị trả về

`true` nếu `haystack` chứa `needle`. Ngược lại, trả về `false`.
