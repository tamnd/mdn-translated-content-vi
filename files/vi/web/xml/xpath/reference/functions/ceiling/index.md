Hàm `ceiling` đánh giá một số thập phân và trả về số nguyên nhỏ nhất lớn hơn hoặc bằng số thập phân đó.

## Cú pháp

```plain
ceiling( number )
```

### Tham số

- `number`
  - : Số cần được đánh giá.

### Giá trị trả về

Số nguyên gần nhất lớn hơn hoặc bằng `number`.

ví dụ:

`ceiling (5.2)` = _6_

`ceiling (-5.2)` = _-5_
