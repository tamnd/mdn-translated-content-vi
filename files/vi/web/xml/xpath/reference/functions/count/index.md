Hàm `count` đếm số nút trong một node-set và trả về một số nguyên.

## Cú pháp

```plain
count( node-set )
```

### Tham số

- `node-set`
  - : Node-set cần được đếm.

### Giá trị trả về

Một số nguyên biểu thị số lượng nút trong node-set.
