Hàm `normalize-space` loại bỏ khoảng trắng ở đầu và cuối khỏi một chuỗi, thay thế các chuỗi ký tự khoảng trắng bằng một khoảng trắng đơn, rồi trả về chuỗi kết quả.

## Cú pháp

```plain
normalize-space( [string] )
```

### Tham số

- `string` (tùy chọn)
  - : Chuỗi cần được chuẩn hóa. Nếu bỏ qua, chuỗi được dùng sẽ là cùng nút ngữ cảnh được chuyển thành chuỗi.

### Giá trị trả về

Chuỗi đã được chuẩn hóa.
