Hàm `string-length` trả về một số bằng với số ký tự trong một chuỗi đã cho.

## Cú pháp

```plain
string-length( [string] )
```

### Tham số

- `string` (tùy chọn)
  - : Chuỗi cần đánh giá. Nếu bỏ qua, chuỗi được dùng sẽ là cùng nút ngữ cảnh được chuyển thành chuỗi.

### Giá trị trả về

Một số nguyên bằng với số ký tự trong chuỗi.
