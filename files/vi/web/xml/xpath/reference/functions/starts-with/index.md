Hàm `starts-with` kiểm tra xem chuỗi thứ nhất có bắt đầu bằng chuỗi thứ hai hay không và trả về true hoặc false.

## Cú pháp

```plain
starts-with(haystack, needle)
```

### Tham số

- `haystack`
  - : Chuỗi cần kiểm tra.
- `needle`
  - : Chuỗi cần tìm.

### Giá trị trả về

`true` nếu `haystack` bắt đầu bằng `needle`. Ngược lại, `false`.
