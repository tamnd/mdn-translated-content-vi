Thuộc tính **`unicode-bidi`** chỉ định cách việc tích lũy ảnh nền được quản lý.

> [!NOTE]
> Là một thuộc tính trình bày, `unicode-bidi` cũng có một thuộc tính CSS tương ứng: {{cssxref("unicode-bidi")}}. Khi cả hai được chỉ định, thuộc tính CSS sẽ được ưu tiên.

## Phần tử

Bạn có thể dùng thuộc tính này với các phần tử SVG sau:

- {{SVGElement("textPath")}}
- {{SVGElement("text")}}
- {{SVGElement("tspan")}}
