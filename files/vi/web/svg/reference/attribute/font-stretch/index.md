{{deprecated_header}}

Thuộc tính **`font-stretch`** chỉ ra mức độ co hẹp hoặc mở rộng mong muốn trong các glyph được dùng để hiển thị văn bản.

> [!NOTE]
> Với vai trò là một presentation attribute, `font-stretch` cũng có thuộc tính CSS tương ứng: {{cssxref("font-stretch")}}. Khi cả hai đều được chỉ định, thuộc tính CSS sẽ được ưu tiên.

Bạn có thể dùng thuộc tính này với các phần tử SVG sau:

- {{SVGElement("text")}}
- {{SVGElement("textPath")}}
- {{SVGElement("tspan")}}
