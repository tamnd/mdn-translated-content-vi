Thuộc tính **`spacing`** cho biết user agent nên xác định khoảng cách giữa các ký tự kiểu chữ được hiển thị dọc theo một đường dẫn như thế nào.

Bạn có thể dùng thuộc tính này với phần tử SVG sau:

- {{SVGElement("textPath")}}

## Ghi chú sử dụng

<table class="properties">
  <tbody>
    <tr>
      <th scope="row">Giá trị</th>
      <td><code>auto</code> | <code>exact</code></td>
    </tr>
    <tr>
      <th scope="row">Giá trị mặc định</th>
      <td><code>exact</code></td>
    </tr>
    <tr>
      <th scope="row">Có thể animate</th>
      <td>Có</td>
    </tr>
  </tbody>
</table>

- `auto`
  - : Giá trị này cho biết user agent nên dùng các thuật toán bố cục văn bản trên đường dẫn để điều chỉnh khoảng cách giữa các ký tự kiểu chữ nhằm đạt được kết quả trực quan hài hòa.
- `exact`
  - : Giá trị này cho biết các ký tự kiểu chữ nên được hiển thị chính xác theo các quy tắc khoảng cách được chỉ định bởi quy tắc bố cục cho văn bản trên đường dẫn.
