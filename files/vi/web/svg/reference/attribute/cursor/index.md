Thuộc tính `cursor` chỉ định con trỏ chuột được hiển thị khi con trỏ chuột ở trên một phần tử.

> [!NOTE]
> Với vai trò là một presentation attribute, `cursor` cũng có thuộc tính CSS tương ứng: {{cssxref("cursor")}}. Khi cả hai đều được chỉ định, thuộc tính CSS sẽ được ưu tiên.
