Thuộc tính [toàn cục](/en-US/docs/Web/HTML/Reference/Global_attributes) **`part`** chứa danh sách tên các phần (part names) của phần tử, phân cách nhau bởi dấu cách. Tên phần cho phép CSS chọn và tạo kiểu cho các phần tử cụ thể trong một shadow tree thông qua phần tử giả {{CSSxRef("::part")}}.

Xem [thuộc tính `exportparts`](/en-US/docs/Web/HTML/Reference/Global_attributes/exportparts) để biết ví dụ sử dụng.
