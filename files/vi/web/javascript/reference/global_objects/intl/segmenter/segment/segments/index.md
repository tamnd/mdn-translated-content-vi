Đối tượng **`Segments`** là một tập hợp có thể lặp các phân đoạn của một chuỗi văn bản. Nó được trả về bởi lời gọi phương thức [`segment()`](/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/Segmenter/segment) của một đối tượng [`Intl.Segmenter`](/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/Segmenter).

## Phương thức instance

- [`Segments.prototype.containing()`](/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/Segmenter/segment/Segments/containing)
  - : Trả về một đối tượng mô tả phân đoạn trong chuỗi gốc chứa code unit tại chỉ số được chỉ định.
- [`Segments.prototype[Symbol.iterator]()`](/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/Segmenter/segment/Segments/Symbol.iterator)
  - : Trả về một iterator để duyệt qua các phân đoạn.
