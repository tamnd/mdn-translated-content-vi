**`emulation`** module chứa các lệnh để ghi đè hành vi của một số Web API nhất định, chẳng hạn như [Geolocation API](/en-US/docs/Web/API/Geolocation_API) và [Screen Orientation API](/en-US/docs/Web/API/Screen_Orientation_API), nhằm mô phỏng các môi trường khác nhau.
