Các lệnh WebDriver classic được gửi dưới dạng yêu cầu HTTP đến các endpoint cụ thể trên đầu xa.
Các lệnh được dùng để điều khiển hành vi của user agent, khám phá và thao tác các phần tử DOM, cũng như quản lý cửa sổ và phiên trình duyệt.

Khi một WebDriver client gửi lệnh, nó chỉ định phương thức HTTP (`GET`, `POST`, `DELETE`) và đường dẫn endpoint tương ứng với thao tác mong muốn.
Đầu xa xử lý lệnh và trả về phản hồi HTTP chứa kết quả hoặc lỗi.
