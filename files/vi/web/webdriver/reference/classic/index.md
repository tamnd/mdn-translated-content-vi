WebDriver classic là giao thức WebDriver nguyên bản, cung cấp các lệnh dựa trên HTTP để tự động hóa trình duyệt.
Phần tham khảo này mô tả các lệnh và timeout có sẵn trong WebDriver classic.

- [Commands](/en-US/docs/Web/WebDriver/Reference/Classic/Commands)
  - : Các thao tác dựa trên HTTP để kiểm tra và điều khiển hành vi của trình duyệt, quản lý phiên và cửa sổ trình duyệt, cũng như tương tác với các phần tử DOM.
- [Timeouts](/en-US/docs/Web/WebDriver/Reference/Classic/Timeouts)
  - : Các định nghĩa timeout kiểm soát hành vi cho việc chèn script, điều hướng tài liệu và truy xuất phần tử.
