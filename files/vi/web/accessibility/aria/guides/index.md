Accessible Rich Internet Applications (**ARIA**) định nghĩa các cách để làm cho web dễ tiếp cận hơn với người khuyết tật.
Trang này liệt kê các hướng dẫn mà bạn có thể dùng để cải thiện trợ năng của các tính năng trên trang web như bảng, biểu mẫu và điều hướng bằng bàn phím.
