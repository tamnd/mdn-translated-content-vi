Module **CSS ruby layout** cung cấp mô hình hiển thị và các điều khiển định dạng liên quan đến việc hiển thị chú thích ruby. Chú thích ruby là một dạng chú thích xen dòng, bao gồm các chuỗi văn bản ngắn bên cạnh văn bản cơ sở. Chúng thường được dùng trong các tài liệu Đông Á để chỉ cách phát âm hoặc định nghĩa ý nghĩa.

## Tham khảo

### Thuộc tính

- {{cssxref("ruby-align")}}
- {{cssxref("ruby-position")}}

Module CSS ruby layout cũng giới thiệu các thuộc tính `ruby-merge` và `ruby-overhang`. Hiện tại, không có trình duyệt nào hỗ trợ các tính năng này.

### Giá trị display

Module CSS ruby layout thêm các giá trị sau vào thuộc tính {{cssxref("display")}}:

- `ruby`
- `ruby-base`
- `ruby-text`
- `ruby-base-container`
- `ruby-text-container`
