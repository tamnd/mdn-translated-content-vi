Phần tử giả **`::-webkit-slider-runnable-track`** trong [CSS](/en-US/docs/Web/CSS) đại diện cho "rãnh trượt" (rãnh mà trong đó con trỏ di chuyển) của phần tử {{HTMLElement("input/range", '&lt;input type="range"&gt;')}}.

## Cú pháp

```css
::-webkit-slider-runnable-track {
  /* ... */
}
```
