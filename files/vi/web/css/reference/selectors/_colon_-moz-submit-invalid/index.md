**`:-moz-submit-invalid`** là [lớp giả](/vi/docs/Web/CSS/Reference/Selectors/Pseudo-classes) [CSS](/vi/docs/Web/CSS) là một [phần mở rộng Mozilla](/vi/docs/Web/CSS/Reference/Mozilla_extensions) đại diện cho bất kỳ {{HTMLElement("button")}} gửi nào trên các biểu mẫu mà nội dung không hợp lệ dựa trên [ràng buộc xác thực](/vi/docs/Web/HTML/Guides/Constraint_validation) của chúng.

Theo mặc định, không có kiểu nào được áp dụng. Bạn có thể sử dụng lớp giả này để tùy chỉnh giao diện của nút gửi khi có các trường biểu mẫu không hợp lệ.

## Cú pháp

```css
:-moz-submit-invalid {
  /* ... */
}
```

Không thuộc bất kỳ tiêu chuẩn nào.
