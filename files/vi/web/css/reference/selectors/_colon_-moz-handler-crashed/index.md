{{Non-standard_header}}

**`:-moz-handler-crashed`** là [lớp giả](/vi/docs/Web/CSS/Reference/Selectors/Pseudo-classes) [CSS](/vi/docs/Web/CSS) là một [phần mở rộng Mozilla](/vi/docs/Web/CSS/Reference/Mozilla_extensions) khớp với các phần tử không thể hiển thị vì plugin chịu trách nhiệm vẽ chúng đã bị crash.

> [!NOTE]
> Bộ chọn này chủ yếu được thiết kế để sử dụng bởi các nhà phát triển theme.

## Cú pháp

```css
:-moz-handler-crashed {
  /* ... */
}
```

## Thông số kỹ thuật

Không thuộc bất kỳ tiêu chuẩn nào.
