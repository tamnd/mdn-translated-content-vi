{{Non-standard_header}}

Phần tử giả **`::-webkit-search-cancel-button`** trong [CSS](/en-US/docs/Web/CSS) đại diện cho một nút (nút "hủy") ở cạnh của phần tử {{HTMLElement("input")}} có `type="search"`, nút này xóa giá trị hiện tại của phần tử {{HTMLElement("input")}}. Nút và phần tử giả này không chuẩn, chỉ được hỗ trợ trong WebKit và Blink, do đó có tiền tố nhà cung cấp. Nút xóa chỉ hiển thị trên các phần tử {{HTMLElement("input")}} tìm kiếm không rỗng.

## Cú pháp

```css
selector::-webkit-search-cancel-button {
  /* ... */
}
```
