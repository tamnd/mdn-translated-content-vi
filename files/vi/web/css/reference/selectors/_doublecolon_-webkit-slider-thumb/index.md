{{non-standard_header}}

Phần tử giả **`::-webkit-slider-thumb`** trong [CSS](/en-US/docs/Web/CSS) đại diện cho "núm kéo" mà người dùng có thể di chuyển trong "rãnh trượt" của phần tử {{HTMLElement("input")}} có `type="range"` để thay đổi giá trị số của nó.

## Cú pháp

```css
::-webkit-slider-thumb {
  /* ... */
}
```
