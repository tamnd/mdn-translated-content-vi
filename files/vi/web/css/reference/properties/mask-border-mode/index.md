Thuộc tính **`mask-border-mode`** trong [CSS](/en-US/docs/Web/CSS) xác định chế độ pha trộn được sử dụng trong [mặt nạ viền](/en-US/docs/Web/CSS/Reference/Properties/mask-border).

## Cú pháp

```css
/* Giá trị từ khóa */
mask-border-mode: luminance;
mask-border-mode: alpha;

/* Giá trị toàn cục */
mask-border-mode: inherit;
mask-border-mode: initial;
mask-border-mode: revert;
mask-border-mode: revert-layer;
mask-border-mode: unset;
```

### Giá trị

- `luminance`
  - : Các giá trị độ sáng của hình ảnh mặt nạ viền được dùng làm giá trị mặt nạ.
- `alpha`
  - : Các giá trị alpha của hình ảnh mặt nạ viền được dùng làm giá trị mặt nạ.
