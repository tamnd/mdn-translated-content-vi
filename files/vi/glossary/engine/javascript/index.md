**Công cụ JavaScript** (JavaScript engine) là các trình thông dịch phân tích cú pháp và thực thi mã {{glossary("JavaScript")}}. Các công cụ JavaScript hiện đại sử dụng biên dịch just-in-time (JIT) để chuyển đổi mã JavaScript thành mã máy có thể được thực thi bởi bộ xử lý của máy tính. Công cụ JavaScript thường được phát triển và sử dụng trong {{glossary("browser", "trình duyệt")}} web để chạy mã phía client nhưng cũng có thể được sử dụng trong các môi trường phía máy chủ như {{glossary("Node.js")}}.

Trong một trình duyệt, công cụ JavaScript hoạt động cùng với công cụ kết xuất thông qua các ràng buộc {{glossary("DOM", "Document Object Model")}} và {{glossary("WebIDL", "Web IDL")}}. Một số công cụ JavaScript cũng thực thi mã {{glossary("WebAssembly")}} trong cùng môi trường sandbox với mã JavaScript thông thường.

Đừng nhầm lẫn công cụ JavaScript với {{glossary("engine/rendering", "công cụ kết xuất")}}, vốn cũng là những phần quan trọng của trình duyệt.

Các công cụ JavaScript phổ biến bao gồm:

- [V8](https://v8.dev/)
- [SpiderMonkey](https://spidermonkey.dev/)
- [JavaScriptCore](https://developer.apple.com/documentation/javascriptcore)
