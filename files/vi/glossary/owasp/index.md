**OWASP** (Open Web Application Security Project — Dự án Bảo mật Ứng dụng Web Mở) là một tổ chức phi lợi nhuận và mạng lưới toàn cầu hoạt động vì bảo mật trong Phần mềm Tự do, đặc biệt là trên Web.
