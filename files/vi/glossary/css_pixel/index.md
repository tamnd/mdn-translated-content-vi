Thuật ngữ **pixel CSS** đồng nghĩa với đơn vị độ dài tuyệt đối CSS _px_ — được [định nghĩa chuẩn](/en-US/docs/Web/CSS/Guides/Values_and_units/Numeric_data_types#absolute_length_units) là chính xác 1/96 của 1 inch CSS (_in_).
