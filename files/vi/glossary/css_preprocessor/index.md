**Bộ tiền xử lý CSS** (CSS preprocessor) là một chương trình cho phép bạn tạo ra {{Glossary("CSS")}} từ cú pháp {{Glossary("syntax","riêng")}} của bộ tiền xử lý.

Có nhiều bộ tiền xử lý CSS để lựa chọn, tuy nhiên hầu hết các bộ tiền xử lý CSS sẽ thêm một số tính năng không có trong CSS thuần túy, như mixin, bộ chọn lồng nhau, bộ chọn kế thừa, v.v. Các tính năng này làm cho cấu trúc CSS dễ đọc và dễ bảo trì hơn.

Để sử dụng bộ tiền xử lý CSS, bạn phải cài đặt trình biên dịch CSS trên {{Glossary("server","máy chủ")}} web của mình, hoặc sử dụng bộ tiền xử lý CSS để biên dịch trên môi trường phát triển, sau đó tải tệp CSS đã biên dịch lên máy chủ web.
