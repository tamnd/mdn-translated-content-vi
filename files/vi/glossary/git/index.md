**Git** là hệ thống Quản lý Mã Nguồn ({{Glossary("SCM")}}) phân tán, miễn phí và mã nguồn mở. Git giúp quản lý codebase với các nhóm phát triển phân tán. Điểm khác biệt so với các hệ thống SCM trước đây là khả năng thực hiện các thao tác thông thường (tạo nhánh, commit, v.v.) ngay trên máy phát triển cục bộ mà không cần thay đổi kho lưu trữ chính hay thậm chí có quyền ghi vào đó.
