Java là một ngôn ngữ {{glossary("Compile","biên dịch")}}, {{glossary("OOP","hướng đối tượng")}}, có tính di động cao trong số các {{Glossary("computer programming","ngôn ngữ lập trình")}}.

Java có kiểu tĩnh và có cú pháp tương tự C. Nó đi kèm với một thư viện lớn các hàm sẵn sàng sử dụng, được gọi là Java Software Development Kit (SDK).

Các chương trình được {{glossary("Compile","biên dịch")}} một lần trước thời gian chạy thành định dạng bytecode và gói độc quyền chạy trong Máy ảo Java (JVM). JVM có sẵn trên nhiều nền tảng, cho phép các chương trình Java chạy ở hầu hết mọi nơi mà không cần biên dịch hoặc đóng gói lại. Điều này làm cho nó trở thành ngôn ngữ được ưa thích trong nhiều doanh nghiệp lớn với môi trường đa dạng, nhưng có thể bị coi là "nặng nề".
