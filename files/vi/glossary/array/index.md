**Mảng** (array) là một tập hợp dữ liệu có thứ tự ({{Glossary("primitive", "kiểu nguyên thủy")}} hoặc {{Glossary("object", "đối tượng")}} tùy thuộc vào ngôn ngữ). Mảng được dùng để lưu trữ nhiều giá trị trong một tên biến duy nhất. Ngược lại, một biến thông thường chỉ có thể lưu trữ một giá trị.

Mỗi mục trong mảng có một số gắn liền với nó, được gọi là chỉ số số nguyên, cho phép bạn truy cập vào nó. Trong JavaScript, mảng bắt đầu từ chỉ số 0 và có thể được thao tác với nhiều {{Glossary("Method", "phương thức")}} khác nhau.

Mảng trong JavaScript trông như thế này:

```js
// Arrays in JavaScript can hold different types of data
const myArray = [1, 2, 3, 4];
const barbieDollNamesArray = ["Barbie", "Ken", "Midge", "Allan", "Skipper"];

// Array indexes starts at 0.
console.log(myArray[0]); // output: 1
console.log(barbieDollNamesArray[2]); // output: "Midge"
```
