**Secure context** (ngữ cảnh bảo mật) là `Window` hoặc `Worker` đáp ứng các tiêu chuẩn tối thiểu nhất định về xác thực và bảo mật. Nhiều Web API và tính năng chỉ có thể truy cập được trong các ngữ cảnh bảo mật, giúp giảm cơ hội bị mã độc hại lợi dụng.

Để biết thêm thông tin, xem: [Web > Security > Secure Contexts](/en-US/docs/Web/Security/Defenses/Secure_Contexts).
