**Responsive Web Design** (**RWD** - Thiết kế web đáp ứng) là khái niệm trong phát triển web tập trung vào việc làm cho các trang web hiển thị và hoạt động tốt nhất trên mọi thiết bị máy tính cá nhân, từ máy tính để bàn cho đến điện thoại di động.
