JavaScript (hay "JS") là ngôn ngữ lập trình được sử dụng phổ biến nhất cho các script phía máy khách (client-side) trên trang web, nhưng nó cũng thường xuyên được dùng ở phía {{Glossary("Server","máy chủ")}}, sử dụng runtime như {{Glossary("Node.js")}}, {{Glossary("Deno")}}, và [Bun](https://bun.com/).

JavaScript **không nên** bị nhầm lẫn với [ngôn ngữ lập trình Java](<https://en.wikipedia.org/wiki/Java_(programming_language)>). Mặc dù _"Java"_ và _"JavaScript"_ đều là thương hiệu (hoặc thương hiệu đã đăng ký) của Oracle tại Mỹ và các quốc gia khác, hai ngôn ngữ lập trình này có cú pháp, ngữ nghĩa và trường hợp sử dụng khác nhau đáng kể.

JavaScript chủ yếu được sử dụng trong trình duyệt, cho phép lập trình viên thao tác nội dung trang web qua {{Glossary("DOM")}}, truy xuất nội dung từ máy chủ bằng API {{domxref("Window/fetch", "fetch()")}}, lưu trữ dữ liệu phức tạp bằng {{Glossary("IndexedDB")}}, vẽ đồ họa với {{Glossary("canvas")}}, tương tác với thiết bị đang chạy trình duyệt qua nhiều {{Glossary("API","API")}} khác nhau, và nhiều hơn nữa. JavaScript là một trong những ngôn ngữ được sử dụng phổ biến nhất trên thế giới, nhờ sự tăng trưởng và cải thiện hiệu năng gần đây của các {{Glossary("API","API")}} có trong trình duyệt.

## Nguồn gốc và lịch sử

Được hình thành như một ngôn ngữ phía máy chủ bởi Brendan Eich (khi đó làm việc cho Netscape Corporation), JavaScript sớm xuất hiện trên Netscape Navigator 2.0 vào tháng 9 năm 1995. JavaScript đã gặt hái thành công ngay lập tức và {{glossary("Microsoft Internet Explorer", "Internet Explorer 3.0")}} giới thiệu hỗ trợ JavaScript dưới tên JScript vào tháng 8 năm 1996.

Vào tháng 11 năm 1996, Netscape bắt đầu hợp tác với Ecma International để biến JavaScript thành một tiêu chuẩn công nghiệp. Kể từ đó, JavaScript được chuẩn hóa được gọi là ECMAScript và được quy định theo ECMA-262. Tiêu chuẩn này liên tục được cập nhật và triển khai (một _tiêu chuẩn sống_).

Gần đây, sự phổ biến của JavaScript đã mở rộng hơn nữa thông qua nền tảng [Node.js](https://nodejs.org/en) thành công — môi trường runtime JavaScript đa nền tảng phổ biến nhất bên ngoài trình duyệt. Node.js — được xây dựng bằng [V8 JavaScript Engine của Chrome](<https://en.wikipedia.org/wiki/V8_(JavaScript_engine)>) — cho phép lập trình viên sử dụng JavaScript như một ngôn ngữ script để tự động hóa công việc trên máy tính và xây dựng các máy chủ {{Glossary("HTTP")}} và {{Glossary("WebSockets")}} đầy đủ tính năng.
