eXtensible Markup Language (XML) là ngôn ngữ đánh dấu chung được W3C quy định. Ngành công nghệ thông tin (IT) sử dụng nhiều ngôn ngữ dựa trên XML làm ngôn ngữ mô tả dữ liệu.

Thẻ XML trông giống thẻ HTML, nhưng XML linh hoạt hơn nhiều vì nó cho phép người dùng định nghĩa thẻ của riêng họ. Theo cách này, XML hoạt động như một siêu ngôn ngữ — tức là nó có thể được sử dụng để định nghĩa các ngôn ngữ khác, chẳng hạn như {{Glossary("RSS")}}. Hơn nữa, HTML là ngôn ngữ trình bày, trong khi XML là ngôn ngữ mô tả dữ liệu. Điều này có nghĩa XML có các ứng dụng rộng hơn nhiều so với chỉ Web. Ví dụ, dịch vụ web có thể sử dụng XML để trao đổi yêu cầu và phản hồi.
