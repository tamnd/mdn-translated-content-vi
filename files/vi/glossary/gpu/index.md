GPU (Graphics Processing Unit - Đơn vị xử lý đồ họa) là một thành phần máy tính tương tự như CPU (Central Processing Unit - Đơn vị xử lý trung tâm). Nó chuyên xử lý việc vẽ đồ họa (cả 2D và 3D) lên màn hình của bạn.
