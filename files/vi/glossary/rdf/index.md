**RDF** (Resource Description Framework — Khung Mô tả Tài nguyên) là ngôn ngữ do W3C phát triển để biểu diễn thông tin trên World Wide Web, chẳng hạn như các trang Web. RDF cung cấp cách mã hóa tiêu chuẩn thông tin tài nguyên để nó có thể được trao đổi hoàn toàn tự động giữa các ứng dụng.
