WebExtensions là hệ thống đa trình duyệt để phát triển tiện ích trình duyệt trong Firefox. Hệ thống này cung cấp các API được hỗ trợ rộng rãi trên các trình duyệt khác nhau như Mozilla Firefox, Google Chrome, Opera Browser, Microsoft Edge, hoặc Apple Safari.
