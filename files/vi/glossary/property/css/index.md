Một **thuộc tính CSS** là một đặc tính (như màu sắc) mà giá trị kết hợp với nó xác định một khía cạnh về cách trình duyệt hiển thị phần tử đó.

Đây là ví dụ về một quy tắc CSS:

```css
/* "div" là bộ chọn chỉ ra rằng tất cả các phần tử div */
/* trong tài liệu sẽ được áp dụng quy tắc này */
div {
  /* Thuộc tính "color" với giá trị "black" cho biết */
  /* văn bản sẽ có màu đen */
  color: black;

  /* Thuộc tính "background-color" với giá trị "white" cho biết */
  /* màu nền của các phần tử sẽ là màu trắng */
  background-color: white;
}
```
