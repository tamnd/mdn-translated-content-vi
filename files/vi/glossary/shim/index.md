**Shim** là đoạn code được dùng để sửa chữa hành vi của code đã tồn tại, thường bằng cách thêm API mới để khắc phục vấn đề. Điều này khác với {{Glossary("polyfill")}}, vốn triển khai một API mới chưa được trình duyệt mặc định hỗ trợ.
