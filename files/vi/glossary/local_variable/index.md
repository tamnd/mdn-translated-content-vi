Một {{glossary("variable","biến")}} mà tên của nó chỉ được liên kết với {{glossary("value","giá trị")}} của nó trong một {{Glossary("local scope","phạm vi cục bộ")}}.

## Ví dụ

```js
let global = 5; // A global variable

function fun() {
  let local = 10; // A local variable
}
```
