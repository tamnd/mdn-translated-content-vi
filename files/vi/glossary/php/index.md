PHP (từ viết tắt đệ quy của PHP: Hypertext Preprocessor) là ngôn ngữ kịch bản phía máy chủ mã nguồn mở, có thể được nhúng vào HTML để xây dựng các ứng dụng web và trang web động.

## Ví dụ

### Cú pháp cơ bản

```php
// bắt đầu mã PHP
<?php
  // mã PHP ở đây
?>
// kết thúc mã PHP
```

### In dữ liệu ra màn hình

```php
<?php
  echo "Hello World!";
?>
```

### Biến PHP

```php
<?php
  // biến
  $name='Danilo';
  $surname='Santos';
  $country='Brasil';
  $email='danilocarsan@gmail.com';

  // in các biến
  echo $name;
  echo $surname;
  echo $country;
  echo $email;
?>
```
