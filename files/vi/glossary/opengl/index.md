**OpenGL** (**Open Graphics Library** — Thư viện Đồ họa Mở) là một giao diện lập trình ứng dụng (API) đa ngôn ngữ, đa nền tảng để kết xuất đồ họa vector 2D và 3D. API này thường được dùng để tương tác với bộ xử lý đồ họa (GPU), giúp thực hiện kết xuất được tăng tốc bởi phần cứng.
