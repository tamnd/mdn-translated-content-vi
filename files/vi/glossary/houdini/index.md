Houdini là tên viết tắt của [CSS-TAG Houdini Task Force](https://github.com/w3c/css-houdini-drafts/wiki/), hoạt động từ năm 2015 đến 2019, và là tên của tập hợp các [API JavaScript CSS cấp thấp và các đề xuất đặc tả](https://drafts.css-houdini.org/) phát sinh từ nhóm công tác đó, một số trong đó đã được triển khai.

Mục tiêu của Houdini Task Force là cung cấp cho các nhà phát triển khả năng tạo ra các tính năng CSS mới mà không cần phải chờ đợi các đặc tả tiêu chuẩn web xác định chúng và trình duyệt triển khai chúng. Điều này đạt được bằng cách đề xuất các tính năng mở rộng CSS thông qua việc kết nối vào quá trình tạo kiểu và bố cục của công cụ kết xuất trình duyệt. Các đề xuất Houdini cho phép các nhà phát triển truy cập vào [CSS Object Model](/en-US/docs/Web/API/CSS_Object_Model) ({{Glossary("CSSOM")}}), cho phép các nhà phát triển viết mã mà trình duyệt có thể phân tích cú pháp như CSS.

Một tiền đề chính của các tính năng Houdini là cung cấp hiệu suất tốt hơn so với các tính năng JavaScript tương đương bằng cách để trình duyệt tạo CSSOM trực tiếp, bao gồm các quá trình bố cục, vẽ và tổng hợp.

Các Houdini API bao gồm [CSS Painting API](/en-US/docs/Web/API/CSS_Painting_API), [CSS Properties and Values API](/en-US/docs/Web/API/CSS_Properties_and_Values_API), [CSS Typed Object Model API](/en-US/docs/Web/API/CSS_Typed_OM_API), và [Worklets](/en-US/docs/Web/API/Worklet).
